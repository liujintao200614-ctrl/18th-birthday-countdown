# 18岁生日倒计时 🎂

一个可爱的生日倒计时网页,支持多主题切换、姓名自定义、点击撒花等互动效果。

## 功能

- 实时倒计时(天/时/分/秒)
- 4 套主题配色(草莓奶油 / 星黛紫梦 / 蜜桃汽水 / 薄荷软糖)
- 可点击编辑的名字(自动保存到浏览器本地)
- 点击蛋糕撒花特效
- 生日进度条
- 倒计时结束后自动切换为庆祝页面

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 部署

本项目是纯静态页面,可直接部署到 [Vercel](https://vercel.com):

1. Fork / clone 本仓库
2. 在 Vercel 中 “Add New Project” 并导入该仓库
3. Framework Preset 选择 “Other”,无需构建命令,直接部署即可
